# 🛒 Shopping Cart (E-Commerce Web Application)

## 📌 Project Overview
This is a full-stack E-Commerce web application developed using Java and Spring Boot.  
The application allows users to browse products, manage their cart, and provides an admin panel to manage the platform efficiently.

---

## 🚀 Features

### 👤 User Features
- 🔐 User Registration & Login
- 🛍️ Browse Products
- 🛒 Add to Cart
- ➕ Update Quantity / Remove Items
- 📦 Order Management (Basic)
- 👤 User Session Handling

### 🛠️ Admin Features
- ➕ Add New Products
- ✏️ Update Product Details
- ❌ Delete Products
- 📋 View All Products
- 👥 Manage Users (if implemented)
- 📦 Manage Orders (if implemented)

---

## 🛠️ Technologies Used
- **Backend:** Java, Spring Boot
- **Frontend:** Thymeleaf, HTML, CSS, Bootstrap,JavaScript
- **Database:** MySQL
- **Tools:** Git, GitHub, Maven
